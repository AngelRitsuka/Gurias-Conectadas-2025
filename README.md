# Gurias-Conectadas-2025
Em 2025, tive o privilégio de ensinar duas turmas de jovens mulheres a mesclarem tecnologia e moda para criarem diversas inovações no mundo do trabalho. 
As mesmas aprenderam a primeiro a usar o raciocínio lógico para solucionar problemas comuns na vida cotidiana. Depois passaram a usar diversos tipos de tecnologia para criarem produtos físicos e digitais com modelagem digital, inteligência artificial, programação, design e muito mais. 
Para visualizar os trabalhos  feitos pelas meninas, acesse os murais adicionados.
